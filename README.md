# Heart Stone (마음의 돌)

**마음의 무게를 견뎌내기 위한 심리 웨이트 트레이닝 서비스**

이 프로젝트는 사용자의 심리 상태를 분석하고, 그에 맞는 맞춤형 에세이와 '마음 PT(퍼스널 트레이닝)' 계획을 제공하는 웹 애플리케이션입니다.

## 🌟 주요 기능

*   **마음 인바디 검사**: 50가지 문항을 통해 심리적 과부하, 민감도, 환경 압박 등을 분석합니다.
*   **분석 결과 리포트**:
    *   **심리 에세이**: 현재 상태를 깊이 있게 공감하고 해석해주는 장문의 에세이 ("오늘의 돌, 오늘의 강").
    *   **마음 PT 플랜**: 워밍업부터 응급 상황 대처법까지, 실행 가능한 구체적인 행동 지침(Action Plan) 제공.
*   **몰입감 있는 UI/UX**: 깊은 강을 연상시키는 배경과 부드러운 인터랙션으로 사용자가 검사에 집중할 수 있도록 설계.

## 🛠️ 기술 스택

*   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
*   **Deployment**: Render (Static Site)

## 📂 프로젝트 구조

```text
/
├── index.html      # 메인 진입점
├── style.css       # 전체 스타일링 (반응형, 애니메이션)
├── script.js       # 앱 로직 (상태 관리, 결과 렌더링)
├── data.js         # 질문 및 결과 데이터셋
└── assets/         # 이미지 리소스
```

## 🚀 배포 방법 (Render)

1. 이 저장소(Repository)를 GitHub에 포크하거나 푸시합니다.
2. [Render](https://render.com)에 로그인합니다.
3. **New +** 버튼을 누르고 **Static Site**를 선택합니다.
4. GitHub 저장소를 연결합니다.
5. **Build Command**는 비워둠, **Publish Directory**는 `.` (루트)로 설정합니다.
6. **Create Static Site**를 클릭하면 배포가 완료됩니다.
